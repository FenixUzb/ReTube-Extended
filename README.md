# ReTube Extended

[![Версия](https://img.shields.io/badge/версия-1.6.0--dev-blue.svg)](https://github.com/FenixUzb/ReTube-Extended)
[![Лицензия](https://img.shields.io/badge/лицензия-MIT-green.svg)](LICENSE)
[![Платформа](https://img.shields.io/badge/платформа-Tampermonkey%20%7C%20Violentmonkey-orange.svg)]()

**ReTube Extended** — это мощный и легковесный юзерскрипт (UserScript) для браузеров, предназначенный для значительного улучшения интерфейса и расширения возможностей YouTube и YouTube Music. Скрипт является расширенным форком проекта ReTube от Eject37.

Скрипт написан на чистом JavaScript (vanilla ES6+) без внешних зависимостей и тяжелых фреймворков, что обеспечивает максимальную скорость работы.

---

## 🚀 Основные возможности

Скрипт содержит **более 40 уникальных функций**, которые можно индивидуально включать или отключать через удобное встроенное меню настроек (клавиша `F2`).

## 🛠 Установка

Для работы скрипта вам понадобится менеджер юзерскриптов для браузера.

1.  Установите расширение **Tampermonkey** или **Violentmonkey** для вашего браузера:
    *   [Tampermonkey для Chrome / Edge / Opera / Firefox](https://www.tampermonkey.net/)
    *   [Violentmonkey](https://violentmonkey.github.io/)
2.  Откройте прямую ссылку на скрипт:
    *   **[Установить ReTube Extended](https://raw.githubusercontent.com/FenixUzb/ReTube-Extended/main/ReTube%20Extended.js)** (кликните, чтобы открыть установщик расширения)
3.  Нажмите кнопку **Установить** (Install) в открывшемся окне расширения.

### 📺 Управление и плеер
*   **Принудительное качество видео:** Автоматическая установка разрешения от 144p до 8K (4320p) при запуске.
*   **Откат при паузе:** Автоматический откат видео на 1-5 секунд назад при приостановке воспроизведения (помогает не терять контекст).
*   **Умная прокрутка громкости:** Регулировка громкости колесом мыши с шагом в 1% (удерживая `Shift`).
*   **Быстрая регулировка скорости:** Изменение скорости воспроизведения колесом мыши прямо над кнопкой «Настройки» с шагом 0.1x (правый клик возвращает стандартную скорость).
*   **Принудительная громкость и скорость:** Задание фиксированной громкости и скорости воспроизведения по умолчанию для всех новых видео.
*   **Горячие клавиши:** Делает клавиши управления плеером всегда активными, даже если фокус ввода находится вне плеера.
*   **Копирование ссылки с таймкодом:** Клик по таймеру в плеере копирует ссылку с текущей меткой времени в буфер обмена.

### 🛠 Умные функции и автоматизация
*   **Вернуть дизлайки:** Интеграция с Return YouTube Dislike API для отображения количества дизлайков и соотношения лайков/дизлайков.
*   **Закладки с таймкодами (клавиша `B`):** Возможность сохранять интересные моменты видео. Список закладок отображается справа от плеера.
*   **Скриншот кадра видео (Shift+S):** Сохранение текущего кадра в файл, копирование в буфер обмена или оба действия одновременно.
*   **Пропуск интро и аутро:** Распознавание глав видео и автоматический (или по кнопке) пропуск вступлений и концовок.
*   **Автолайк:** Автоматическая отправка лайка после просмотра определенного процента видео (30%, 50%, 70%, 80%, 90%).
*   **Бейдж кодека и битрейта:** Отображение реального разрешения, используемого кодека (AV1, VP9, H.264) и битрейта в углу плеера.
*   **Скачивание превью:** Кнопка в плеере для быстрого скачивания обложки видео в максимальном разрешении.
*   **Реверс порядка плейлиста:** Кнопка для изменения порядка воспроизведения видео в текущем плейлисте на противоположный.
*   **Сохранение прогресса видео:** Автоматическое продолжение просмотра с того же места при перезагрузке страницы.
*   **Отключение засыпания плеера:** Запрещает автоматическую приостановку воспроизведения при отсутствии активности пользователя (удобно для фонового прослушивания музыки).

### 🎨 Интерфейс и оформление
*   **Плавные анимации:** Мягкое появление элементов страницы при загрузке.
*   **Перекраска YouTube (Темы):** Набор готовых тем (ReTube, ReTube Dark, Тёмный, Пурпурный, Зелёный) и режим «Свои цвета» для полной кастомизации палитры.
*   **Ubuntu Font:** Возможность заменить стандартный шрифт YouTube на более читаемый шрифт Ubuntu.
*   **Точная дата и время загрузки:** Отображение полной даты и времени публикации прямо в названии видео.
*   **Возраст публикации цветом:** Отображение цветного бейджа на превью видео в зависимости от его возраста (зелёный — менее 7 дней, красный — более 1 года).
*   **Помечать просмотренные:** Наглядное выделение видео, которые вы уже посмотрели.
*   **Полные заголовки:** Отключение обрезания названий видео троеточием — заголовки показываются полностью.
*   **Прогресс в заголовке вкладки:** Отображение процента просмотра видео прямо в названии вкладки браузера (например, `[45%] Название видео`).

### 🧹 Очистка мусора и Shorts
*   **Скрытие мусорных элементов:** Возможность скрыть ненужные кнопки плеера, кнопки «Поделиться», «Скачать», «Спасибо» под видео, а также элементы поиска.
*   **Скрытие Shorts:** Полное удаление разделов Shorts с главной страницы, подписок, поиска и боковой панели.
*   **Перенаправление Shorts:** Автоматический редирект видео из формата Shorts в стандартный плеер YouTube.
*   **Блокировка автовоспроизведения трейлера:** Запрещает воспроизведение видео-трейлера при переходе на страницу канала.

### 🔗 Интеграции и технические параметры
*   **Интеграция с Telegram:** Кнопка в плеере для отправки ссылки на видео в Telegram-бот для последующей транскрибации (методом API-сервера или прямой ссылки).
*   **Собственный YouTube Data API v3 ключ:** Возможность указать свой API-ключ для функций получения дат и количества видео (для обхода общих лимитов).
*   **Менеджер кэша:** Встроенный инструмент очистки кэша дат, количества видео, дизлайков и сброса API-ключей.

## ⚙️ Настройка

Чтобы открыть меню настроек ReTube Extended:
1.  Перейдите на сайт [YouTube](https://www.youtube.com) или [YouTube Music](https://music.youtube.com).
2.  Нажмите клавишу **`F2`** на клавиатуре.
3.  Настройте функции под себя и нажмите кнопку **«Сохранить»**.
4.  Для закрытия меню настроек нажмите **`F2`** или клавишу **`Escape`**.

Для вызова быстрой шпаргалки по горячим клавишам ReTube и YouTube нажмите **`Shift + ?`**.
